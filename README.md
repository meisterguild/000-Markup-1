24時間（React環境構築含め）

フローチャートの図がウィンドウサイズを縮めると崩れる…。

## 環境構築
### yarnをインストール
```bash
npm install -g yarn
```

### パッケージのインストール
```bash
yarn install
```

### ローカルサーバーの起動
```bash
yarn start
```
